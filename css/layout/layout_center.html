3.1 块元素水平居中（奇技淫巧）

代码示例： 
          .outer{
            /* 这里用作清除浮动 */
            overflow: hidden;
          }
          .wrap{
            float: left;
            left: 50%;
            /* 为子元素 定位提供据点 */
            position: relative;
          }
          .inner{
            position: relative;
            left: -50%;
          }
          /*
          .wrap 左浮动50%，相当于它的 左边位于 .out 的 中线
          .inner 左浮动-50%，就相当于以.out的中线为基准左移一般，于是就居中了
          .inner 里的position: relative，就是为了 让 .inner 以 .wrap 为基准左移而不是 .out 

          关于这个方法，最大优点就是可以不关注 宽度问题，效果类似  text-align: center;和 display:inline-block;
          */

4. 块元素垂直居中 (writing-mode)

代码示例： 
          .parent {
            /* 修改文档流方向 【 文本水平或垂直排布 】为竖版 */
            writing-mode: vertical-lr;/* vertical-rl 亦可，注意文字方向*/
            height: 200px;
          }

          .child{
            /* 原理同水平方向 margin: 0 auto; */
            margin: auto 0;
            /* 高度必要，否则会充满父级 */
            height: 100px;
          }
          /* 
            通过 writing-mode  修改 文档流 从而使用 `margin: auto 0; ` 可以垂直居中块；
            注意：
            1. 修改文档流后，高度会自动充满 需要配合设置高度；宽度与原来文档流高度一样，内容撑开或者设置固定值
            2. 会导致文字方向发生改变， `text-align`,`line-height`作用方向也会发生对调。
            3. 有副作用，writing-mode属性的可继承性，会导致该元素下得所有子元素全部流方向全部变为横向。而且此时水平方向不能再使用此方法实现居中了
          */

5.水平垂直居中

5.1 position水平垂直居中

5.1.1 position 配合 位移 -- 居中元素宽高固定

代码示例： 
          .parent {
            position: relative;
            width: 300px;
            height: 300px;
          }

          .child {
            width: 150px;
            height: 150px;

            position: absolute;
            /* 最普遍的方式：缺点 需要宽高固定 */
            left: 50%;
            top: 50%;
            /* 
              margin-left: -75px;
              margin-top: -75px; 
            */
            transform: translate(-75px, -75px);
          }
          /* 
          translate固定值 与 margin-left、margin-top 没有本质差别，
          就是进行反向位移处理
           */
